# Тест для Сбербанка
Программа заходит на сайт http://www.sberbank.ru/ru/quotes/converter и выполняет три теста.

Для запуска программы, необходимо в конфигурационном файле прописать путь к драйверу браузера (gecko.driver), который лежит в папке lib.

Используемые технологии:
- JUnit
- Selenium WebDriver
- Yandex.Allure
- Maven
